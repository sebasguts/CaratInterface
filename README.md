## Install

Download carat via

```
https://github.com/lbfm-rwth/carat
```
and compile it via
```
cd carat
make TOPDIR=`pwd` CFLAGS="-O -g -fPIC" CC="gcc"
```

Then compile via
```
./configure
make
```
